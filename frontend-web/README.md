# SIMPUS Frontend (Vanilla HTML/CSS/JS)

Frontend untuk **SIMPUS (Sistem Manajemen Perpustakaan Digital)**.  
Aplikasi ini terintegrasi dengan Backend Laravel melalui **REST API** menggunakan **Bearer Token**.

## Fitur

### Member
- Register & Login
- Dashboard
- Katalog buku (list + search + pagination)
- Detail buku + rekomendasi buku
- Pinjam buku
- Return buku
- Lihat denda + bayar denda (dummy)
- Riwayat peminjaman (hanya milik sendiri)

### Admin (Librarian)
- Login
- Dashboard
- Kelola buku (CRUD)
- Monitor semua transaksi peminjaman (loans)
- **Catatan:** Admin **tidak bisa return** dan **tidak bisa bayar denda** (hanya monitoring)

## Tech Stack
- HTML / CSS / JavaScript (Vanilla)
- Fetch API untuk komunikasi REST
- (Opsional) Live Server / http-server untuk menjalankan local
